# Job Portal System (Spring Boot + MySQL)

A recruiter-friendly Job Portal backend API built with Spring Boot and MySQL.

## Tech Stack
- Java 17
- Spring Boot (Web, JPA, Validation, Security later)
- MySQL
- Maven

## MVP Features
### Candidate
- Register / Login
- Create profile
- Browse jobs
- Apply to jobs
- Track application status

### Recruiter/Admin
- Post jobs (CRUD)
- View applicants for a job
- Update application status

## Database Tables
- users
- candidate_profiles
- jobs
- applications
